Dmitry Zenovich's PEAR Channel Server

Registering the channel: 
pear channel-discover zenovich.github.io/pear
Listing available packages: 
pear remote-list -c zenovich
Installing a package: 
pear install zenovich/package_name
Installing a specific version/stability: 
pear install zenovich/package_name-1.0.0
pear install zenovich/package_name-beta
Receiving updates via a feed: 
https://zenovich.github.io/pear/feed.xml
